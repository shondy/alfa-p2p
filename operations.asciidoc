== Описание
Сервис предназначен для работы с объектом P2P/Transactions.

=== JSON schema

https://github.com/shondy/alfa-p2p/blob/master/transactions-json-schema[JSON schema] 

=== Параметры объекта P2P/Transactions
|===
| Параметр | Тип | Описание

| id
| String
| ID транзакции

| status
| String
| статус транзакции

| amount
| Long
| сумма транзакции

| fee
| Long
| сумма комиссии

| fullAmount
| Long
| сумма транзакции с комиссией

3+| Currency

| currency
| Object
| Информация по валюте основного счета

| href
| String
| Строка запроса метода GET /currencies

| code
| String
| Мнемоника валюты основного счета

| scale
| Integer
| Количество минорных единиц

| date
| String
| Дата совершения транзакции (YYYYMMDDHHMMSSffffff)

| authorizationCode
| String
| Код авторизации в SV

| acsurl
| String
| URL страницы 3D Secure

3+| SenderCard

| senderCard
| Object
| Информация по карте списания

| href
| String
| Строка запроса метода GET /cards/{cardsID} 

| name
| String
| Название карты

| number
| String
| Номер карты

| expirationYear
| Integer
| Год окончания срока действия

| expirationMonth
| Integer
| Месяц окончания срока действия

| cvv
| Integer
| CVV карты

3+| RecipientCard

| recipientCard
| Object
| Информация по карте зачисления

| href
| String
| Строка запроса метода GET /cards/{cardsID} 

| name
| String
| Название карты

| number
| String
| Номер карты

3+| Authorization

| authorization
| Object
| Информация о 3ds-авторизации

| pareq
| String
| Запрос на аутентификацию отправителя

| md 
| String
| Контрольная сумма

| pares
| String
| Подтверждение авторизации карты

|===

==== Получение параметров транзакции на 3-м шаге
....
Дано:
  клиент выполняет транзакцию по P2P переводу.
Когда:
  клиент успешно подтвердил транзакцию с помощью 3ds-авторизации.
Тогда:
  клиенту отображается страница третьего шага с параметрами транзакции (channelId="C2" и transactionId = "0000000000000001").
....

==== Редактирование названия карты
....
Дано:
  клиент кликает на псевдоссылку с названием карты.
Когда:
  вводит новое название карты (name = 0000000000000001)
Тогда:
  сохраняется новое название карты.
....
===== Example request
[source, json]
----
PATCH /cards/0000000000000001 HTTP/1.1
Host: money.alfabank.ru 
Authorization:
Content-Type: 
Content-Length: 
customerId: AHQFFT
applicationId: ac-p2p

{
  "name": "card1"
}
----
===== Example response
[source, json]
----
HTTP/1.1 200 OK
----
